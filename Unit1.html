Hardware Components

The Motherboard - The computers main circuit board, it is the network that all information must pass through in the computer to reach its destination.

CPU (Central Processing Unit) - Located inside the computer case directly on the motherboard. The CPU is the brain of the computer, when you press a key, click the mouse, etc. Its sending instructions to the CPU.

RAM (Random Access Memory) - The systems short-term memory. Stores data temporarily

Hard Drives - Where your software, documents, etc are stored. Long term storage.

Video Card/GPU - The video card is responsible for what you see on the monitor. The GPU is built into the motherboard for playing graphics-intensive games.

Malware

What is Malware? - Software designed to steal data or inflict damage on a computer or software systems.

Types of Malware - Viruses, Worms, Trojan Horse, Spyware, Adware and Ransomware

Viruses - Links malicious code with clean code and waits for the user to execute it

Prevention - Scandrive: Checks your hard drive for errors. Fragmentation: Rearranges files and removes any clogs in the system.
